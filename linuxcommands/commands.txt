Linux commands 

ctrl+alt+T - команда для открытия терминала на Ubuntu
F4+terminal - команда для открытия терминала

~ -> символ домашней директории
/ -> символ перехода с одной директории в другую, еще и обозначает корневую директорию

.. -> обозначает предыдущую директорию
. -> обозначает текущую директорию

1. cd(change directory) - команда для перехода из одной директории в другую

2. ls(list) - команда для вывода содержимого текущей директории

3. mkdir (make directory) - команда для создания директории

4. rmdir (remove directory) - удаляет пустые директории

5. rm -rf - принудительное удаление

6. rm (remove) - удаляет файлы

7. touch - создает указанный файл

8. pwd (print working directory) - выводит абсолютный путь до текущей директории

9. mv (move) - команда для переименовывания файла или директории, а также для перемещения файлов и директорий
	переименовывание: mv john.txt hello.txt
	перемещение: mv john.txt ~/Desktop

10. ls -a -> команда для вывода всех файлов и директорий в текущей директории (как скрытых так и публичных)

11. history - показать все предыдущие команды

12. cat <file name> - выводит в терминал все содержимое файла

13. echo - команда для быстрой записи текстовой информации в какой-либо файл
	echo 'текст для записи' > название_файла

14. nano <path_to_file> - команда открывает файл для редактирования в графическом интерфейсе
	*если нет такого файла, то nano создаст новый файл

15. du -sh test.txt -> выведите в терминал объем занимаемой памяти файла test.txt

16. exit -> закройте терминал при помощи определенной команды

17. cp (copy) - копирование файла или директории
	cp file_name ~/Desktop/...
